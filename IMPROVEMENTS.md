# ALMAENG 모바일 인터페이스 개선 제안

## 🎯 핵심 기능: 동일 성분 함량 대비 가격 비교

### 1. 성분 기반 검색 기능 강화
- **현재**: 제품명/브랜드 검색만 가능
- **개선**: 
  - 성분명으로 검색 (예: "비타민C 1000mg")
  - 성분 함량 범위 검색 (예: "비타민D 1000-2000IU")
  - 복합 성분 검색 (예: "비타민C + 아연")

### 2. 동일 성분 함량 대비 가격 비교 수학 로직
```python
# 핵심 계산식
가격/단위 = 제품가격 / (성분함량 × 총섭취횟수)

예시:
- 제품A: 비타민C 1000mg × 60정 = 15,000원
  → 15,000 / (1000 × 60) = 0.25원/mg
  
- 제품B: 비타민C 1000mg × 30정 = 8,000원
  → 8,000 / (1000 × 30) = 0.27원/mg
  
→ 제품A가 더 가성비 좋음 (낮을수록 좋음)
```

### 3. 모바일 UI/UX 개선 사항

#### 3.1 홈 화면
- ✅ 검색바를 상단 고정 (스크롤해도 보임)
- ✅ 인기 검색어를 스와이프 가능한 카드 형태로 변경
- ✅ 최근 본 제품 히스토리 추가

#### 3.2 상세 페이지
- ✅ 가격 비교 섹션을 상단으로 이동 (중요 정보 우선)
- ✅ 동일 성분 제품 비교를 스와이프 가능한 카드로 표시
- ✅ 가성비 순위 배지 추가 (🏆 Top 1, 🥈 Top 3 등)
- ✅ 가격 변동 그래프 추가 (30일/90일)

#### 3.3 비교 페이지
- ✅ 찜 목록에서 드래그 앤 드롭으로 비교
- ✅ 비교 결과를 카드 형태로 시각화
- ✅ 차이점을 색상으로 강조 (초록=일치, 빨강=차이)

#### 3.4 성분 검색 페이지 (신규)
- ✅ 성분명 입력 → 함량 범위 설정 → 가격 비교
- ✅ 필터: 가격대, 브랜드, 평점
- ✅ 정렬: 가성비순, 가격순, 함량순
