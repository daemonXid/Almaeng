가품 방지 정책

✅ 정품 보장

알맹AI는 공식 API를 통해 정품 정보만 제공합니다.

1. 정품 인증 시스템

  • 쿠팡 파트너스 공식 API 연동
  • 네이버 쇼핑 검색 API 연동
  • 11번가 Open API 연동
  • 모든 상품은 공식 쇼핑몰에서 직접 판매

2. 신뢰할 수 있는 판매자

알맹AI에 표시되는 모든 상품은 다음 쇼핑몰에서 검증된 판매자가 등록한 정품입니다:

  • 쿠팡: 로켓배송, 쿠팡 직매입 우선 표시
  • 네이버 쇼핑: 네이버페이 안전거래
  • 11번가: 11번가 인증 판매자

3. 가품 신고

만약 가품이 의심되는 상품을 발견하시면:

  1. 구매하신 쇼핑몰 고객센터에 즉시 신고
  2. 알맹AI (support@almaeng.com)에도 알려주세요
  3. 해당 판매자 정보를 차단 처리하겠습니다

4. 소비자 보호

알맹AI는 다음을 보장합니다:

  • ✅ 공식 API를 통한 정품 정보만 제공
  • ✅ 사용자 신고 시 즉시 조치
  • ✅ 신뢰할 수 있는 판매자 우선 표시
  • ✅ 투명한 가격 정보 제공

5. 면책 조항

알맹AI는 가격 비교 및 정보 제공 서비스이며, 실제 판매는 각 쇼핑몰에서 이루어집니다. 상품의 진품 여부는 각 쇼핑몰과 판매자가 책임집니다.
